🚀 快速开始指南

第一步：选择测评工具

根据您的需求选择合适的测评工具。我们提供人格测评、能力评估、压力测试等多种专业工具。

  • • 大五人格评估 (agent-big-five-50-complete2.json)
  • • MBTI人格测试 (16种人格类型)
  • • 客服能力评估 (Agent-Customer-Service-50-cn.json)
  • • 银行客服专项 (bankclientBig5.json)

第二步：配置人格类型

选择或配置AI Agent的人格类型。支持16种MBTI人格类型，每种人格都有独特的认知特点和行为模式。

INTJ
ESTJ
ISFJ
ENFP

第三步：使用Claude Code技能

使用我们开发的Claude Code技能进行自动化评估：

claude code -p "测评内容" --system-prompt "人格配置" > output.json

第四步：查看评估报告

生成专业的HTML评估报告，包含详细的人格分析、能力评估和发展建议。

7标签页报告 交互式图表 专业分析

快速命令示例

生成ESTJ人格回答：

cat questionnaire.json | claude code -p "ESTJ人格回答" --system-prompt "你是ESTJ类型..." > estj_response.json

生成评估报告：

cat response.json | claude code -p "生成评估报告" --system-prompt "你是评估报告生成器..." > report.html

⚙️ 技术架构与实现

核心架构

模块化设计

采用模块化架构，支持独立测试和灵活组合。核心模块包括：

  • • 测评引擎 (assessment_engine)
  • • 人格分析器 (personality_analyzer)
  • • 报告生成器 (report_generator)
  • • 数据处理器 (data_processor)
数据处理流程

完整的数据处理管道，从测评输入到报告输出：

问卷输入 → 人格化回答 → 评分分析 → 报告生成 → 输出展示

我们开发的Claude Code技能将复杂的评估流程简化为命令行操作， 让AI Agent的心理评估变得像运行命令一样简单。

questionnaire-responder 技能

专门用于基于人格特征生成问卷回答的技能，支持16种MBTI人格类型。

功能特点：
  • • 人格化问卷回答生成
  • • 16种MBTI人格支持
  • • 专业场景适配
  • • 标准化JSON输出
使用示例：
cat questionnaire.json | claude code -p "ESTJ人格回答" --system-prompt "你是ESTJ类型..."

evaluation-report-generator 技能

专业评估报告生成技能，自动生成7标签页HTML报告，包含详细分析和可视化图表。

报告特色：
  • • 7个标准标签页结构
  • • 交互式图表展示
  • • AI人格实验室品牌集成
  • • 响应式设计
使用示例：
cat response.json | claude code -p "生成评估报告" --system-prompt "你是评估报告生成器..."

技能配置与使用

技能文件结构：
.claude/skills/ ├── questionnaire-responder/ │ └── SKILL.md └── evaluation-report-generator/ └── SKILL.md
快速开始步骤：
  1. 1. 将技能文件复制到 .claude/skills/ 目录
  2. 2. 重启Claude Code以加载技能
  3. 3. 使用 claude code --help 验证技能加载
  4. 4. 开始使用技能进行评估
